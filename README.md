ARM Calc web service client for Python
======================================

armcalcsvc
----------

Consumes the ArmCalc web service.

* Should run in either v2.7+ or v3.5.2+ of Python.

### Unit tests (`test_*.py`) ###

These are test scripts for use with the [unittest] Python module.
